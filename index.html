PL · UTM Link Builder i What is a UTM link?
A normal URL with tags on the end that tell analytics where a visitor came from. Nothing is registered anywhere — the tags are read the moment someone clicks and lands on the site. Works on any site that has an analytics tag.

Format: ?utm_source=…&utm_medium=…&utm_campaign=…

Build UTM-tagged campaign links for any Protocol Labs site — drop a link, pick channels, copy tagged URLs & QR codes. Hover the ⓘ icons for help.
